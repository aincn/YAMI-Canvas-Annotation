# YAMI画布批注

YAMI画布批注是一款用于图片批注、修改意见标注和工程文件保存的 Windows 桌面工具。

## 主要功能

- 导入图片并在画布中排列查看
- 从问题位置拖出箭头并添加批注说明
- 编辑、移动、删除批注
- 调整批注样式、文字大小、文字粗细和箭头线条
- 导出和导入 `.yami` 工程文件
- 支持快捷键粘贴图片、撤回、重做和删除选中对象

## 系统要求

- Windows 10/11 64 位系统

## 开发运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
npm run dist
```
