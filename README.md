## ✨ Features / 主要功能

- 🖼️ **Image import and paste**  
  支持图片拖入、导入和粘贴

- 🧭 **Infinite canvas with zoom and pan**  
  支持无限画布、缩放和拖动画布

- 📝 **Arrow annotations with editable text boxes**  
  支持箭头批注和可编辑文字框

- 🔢 **Automatic annotation numbering**  
  支持批注自动编号，适合多处修改反馈

- 📁 **Project import and export**  
  支持 .yami 工程文件导入与导出

- 📸 **Screenshot workflow**  
  支持截图导入工作流

- 💻 **Installer and portable versions**  
  提供安装版与便携版

## 🚀 Usage / 使用方式

### Import Images / 导入图片

Drag images into the canvas, or press Ctrl + V to paste copied images.

可以将图片拖入画布，也可以使用 Ctrl + V 粘贴复制的图片。

### Add Annotations / 添加批注

Use annotation mode to drag from a target area and create an arrow with a text box.

使用批注模式，从目标位置拖拽即可创建箭头和文字说明。

### Save Projects / 保存工程

Export a .yami project file to save images, annotations, and settings.

可导出 .yami 工程文件，用于保存图片、批注和设置，方便后续继续编辑。
